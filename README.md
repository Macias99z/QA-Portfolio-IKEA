# QA Manual Testing - IKEA Search Functionality

## Project Description

This project presents manual test design documentation for the product search functionality on the IKEA e-commerce platform.

The purpose of this project is to demnonstrate test coverage, boundary analysis and edge case thinking without focusing on defect discovery. 

## Scope

- Search input field
- Search button
- Search results page
- No results behavior

## Out of scope: 

- Payments
- Login
- Product purchase

## Types of testing

- Functional Testing
- Boundary Testing
- Edge Case Testing

## Test Artifacts

- Test Plan
- Test Scenarios
- Test Cases
- Edge Cases
- Test Summary Report

- Manual Testing
